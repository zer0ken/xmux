# CLAUDE.md

This project's canonical guidance for AI coding tools lives in the `AGENTS.md`
files (per-directory Working Notes) and `CONTEXT.md` (the glossary and the
orthogonal-design architecture). Those are the single source of truth.

`CLAUDE.md` exists only because Claude Code reads a file of this name; it holds
no separate rules and defers entirely to `AGENTS.md`. Start at the root
`AGENTS.md`, then read the `AGENTS.md` in each directory you touch, and consult
`CONTEXT.md` for the vocabulary and module seams before editing.

Do not add project guidance here — update `AGENTS.md` / `CONTEXT.md` instead, so
the single source of truth cannot drift.
