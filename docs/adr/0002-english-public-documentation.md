# ADR 0002: English Public Documentation

## Status

Accepted

## Context

xmux is an open source project. Documentation committed to the repository is
part of the public developer experience for contributors, users, and coding
agents.

Some temporary planning or scratch files may exist outside the repository and do
not affect the published project.

## Decision

All documentation committed to the repository must be written in English.

Temporary files outside the repository may use another language.

## Consequences

Working Notes, ADRs, README files, specs, plans, and contributor-facing docs use
English.

Existing committed documentation in another language should be translated before
release or before it becomes part of the public project surface.
